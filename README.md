# Matlab_barchart

1. 条形类图形
1） 直方图
rose 函数 rose(theta,x), 其中，参数theta用于确定每一区间与远点的角度，选项x用于指定区间的划分方式。

每一个区域的高度反映落入该区域三角函数的个数，x:均匀划分的扇形区域（默认20）
